dip::SimpleOutputBuffer class

A simple output buffer implementation.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, assignment and conversion operators
    • Functions

The constructor takes a std::vector< dip::uint8 > by reference. This vector needs to remain in scope wherever the SimpleOutputBuffer object is in scope. The first dip::SimpleOutputBuffer::size bytes of this vector will contain the encoded image data after the image writing function has done its thing.
